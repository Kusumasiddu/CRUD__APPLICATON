# 📝 CRUD Application

## 📌 Project Description
This is a CRUD (Create, Read, Update, Delete) web application developed to perform basic database operations. It allows users to add, view, update, and delete records through a simple and user-friendly interface.

This project demonstrates how data can be managed efficiently using backend logic and server-side operations.

---

## 🚀 Features
- ➕ Create new records
- 📄 View all records
- ✏️ Update existing records
- ❌ Delete records
- 💻 Simple and clean user interface

---

## 🛠️ Technologies Used
- JavaScript (Node.js)
- Express.js
- HTML, CSS
- (Optional: Database if used)

---

## ▶️ How to Run the Project

1. Clone the repository:

git clone https://github.com/Kusumasiddu/CRUD__APPLICATON.git


2. Navigate to project folder:

cd CRUD__APPLICATON


3. Install dependencies:

npm install


4. Run the server:

node server.js


5. Open in browser:

http://localhost:3000


---

## 📂 Project Structure
- `server.js` → Main server file  
- `views/` → UI files  
- `assets/` → Static files (CSS, images)  
- `server/` → Backend logic
